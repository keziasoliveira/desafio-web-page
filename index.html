Surgimento da Internet

A internet surgiu em 1969, quando foi nomeada de ARPANET. A primeira conexão foi registrada entre computadores na universidade de Stanford e de UCLA. Foi feita uma tentativa de enviar a palavra "LOGIN" de um computador ao outro, mas apenas as letras "LO" chegaram.

O que é o HTML?

O HTML, ou HyperText Markup Language, foi criado em 1990 pelo cientista Tim Berners-Lee, enquanto ele trabalhava no Centro Europeu de Pesquisa Nuclear, na Suíça.

O objetivo era criar uma maneira de compartilhar e trocar documentos pesquisas entre cientistas de diferentes universidades, mas ao longo do tempo o HTML foi evoluindo para se tornar a linguagem de marcação padrão para páginas da Web.

O que são Tags?

As tags HTMLsão elementos usados para marcar e formatar os conteúdos textual ou visual em documentos HTML. Cada tag HTML tem um nome e uma sintaxe que determina como ela deve ser aberta e fechada.

Abaixo alguns exemplos de tags utilizadas no HTML:

Tags h1 até h6

A tag h1 define a headline mais importante e a h6 define a menos importante

Tag p

Marca o inicio de um paragrafo e o fim de um paragrafo.

Tag mark

Destaca o que estiver dentro da tag

Tag small

Retorna um texto com fonte em tamanho menor. Pode ser usado para comentarios ou simbolos como o de marcas registradas

Tag i

Retorna o que está dentro da tag em italico

Tag u

Retorna o texto que está dentro da tag sublinhado.

Tag strong

Retorna o que está dentro da tag em negrito

Tag ol

Retorna uma lista ordenada (item1, item2, item 3)

Tag ul

Retorna uma lista não ordenada (.item, , .item, .item)

Tag li

Retorna uma lista de elementos

Tag a

Gera um hiperlink, que ao ser clicado leva a outra página. Seu principal atributo é o href que é usado para fazer referência a uma url externa.

Tag hr

Traça uma linha separatória entre dois textos.

Tag sub

Retorna o texto que está dentro da tag sobrescrito.

Tag sup

Retorna o texto que está dentro da tag subescrito.

Tag blockquote

Usada para destacar uma citação de outra fonte.

Tag font

Usada para definir o tipo, tamanho e cor da fonte.

Tag dell

Destaca um texto que foi excluído. Geralmente aparece como uma linha riscando o citado texto

Tag abbr

Retorna o texto dentro da tag como uma abreviação
